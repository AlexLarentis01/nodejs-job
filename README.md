# Question 1: Fibonacci Sequence

Write a recursive function in Node.js to compute the nth Fibonacci number. The Fibonacci sequence is defined as follows:

- F(0) = 0
- F(1) = 1
- F(n) = F(n-1) + F(n-2) for n > 1
